EJERCICIO 1 - DIFERENTES DIRECTIVAS

En este ejercicio se muestran DOS TIPOS DE DIRECTIVAS. Las estructurales (las que afectan al DOOM) y las de atributos (las que afectan a los atributos de los elementos). En este caso, se muestra el uso de "ngClass" y "ngStyle" como directivas de atributos, y "for", "if/else" y "switch" como directivas estructurales.

  • ESTRUCTURALES

  • if/else sirve para mostrar u ocultar elementos según una condición
  • for sirve para iterar sobre una lista de elementos
  • switch case sirve para mostrar diferentes mensajes según el valor de una variable

  • ATRIBUTOS

  • ngClass sirve para aplicar clases CSS de forma condicional
  • ngStyle sirve para aplicar estilos CSS de forma condicional
  • ngModel sirve para crear enlaces bidireccionales entre el modelo y la vista
